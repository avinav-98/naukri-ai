System Architecture
User flow:
1.	User uploads resume
2.	User links Naukri credentials
3.	User clicks Run Pipeline
Pipeline process:
fetch jobs → rank jobs → auto apply
Pipeline runs asynchronously.
Frontend polls:
GET /api/pipeline-runs/{run_id}
every 4 seconds for status updates:
queued → running → completed / failed
